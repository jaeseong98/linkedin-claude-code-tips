## Usage

사내 Claude Code 플러그인 마켓플레이스를 구축하라. 각 팀원이 만든 Skill 파일(.md)을 사내 GitHub 레포의 skills/ 디렉터리에 PR로 올리고, 슬랙 웹훅으로 신규 스킬 등록 알림을 자동 발송한다. 스킬 파일에는 반드시 name, description, trigger 조건, 사용 예시를 포함해야 한다.
